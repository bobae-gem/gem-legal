# 모니터링 시스템 시작 방법

## 1. 라이브러리 설치 (최초 1회)

```
! cd "C:\Users\user\Desktop\클로드\gem법무실\monitoring_system" && pip install -r requirements.txt
```

## 2. 유튜브 API 키 설정 (선택, 무료)

1. https://console.cloud.google.com 접속
2. 새 프로젝트 생성
3. YouTube Data API v3 활성화
4. 사용자 인증 정보 → API 키 생성
5. config.json의 "youtube_api_key" 에 붙여넣기

API 키 없어도 뉴스+커뮤니티 모니터링은 작동함

## 3. 1회 실행 (수동)

```
! cd "C:\Users\user\Desktop\클로드\gem법무실\monitoring_system" && python main.py
```

## 4. 자동 반복 실행 (스케줄러)

```
! cd "C:\Users\user\Desktop\클로드\gem법무실\monitoring_system" && python main.py --schedule
```

- 유튜브: 10분마다 자동 검색
- 뉴스: 30분마다 자동 검색
- 커뮤니티: 1시간마다 자동 검색

## 5. 결과 확인

실행 후 → http://localhost:8080/업데이터.html

업데이터에서:
- [승인] → 메인 상황실 반영
- [보류] → 나중에 검토
- [삭제] → 변경기록에서 확정 삭제

## 전체 흐름

```
python main.py
      ↓
  키워드 검색 (유튜브 + 뉴스 + 커뮤니티)
      ↓
  AI 분류 (8가지 유형)
      ↓
  위험도 판단 (낮음~매우높음)
      ↓
  증거보존 (evidence_storage/ 자동 저장)
      ↓
  dashboard_pending.json 에 전달
      ↓
  업데이터.html 에서 대표 승인
      ↓
  메인 상황실 반영
```
