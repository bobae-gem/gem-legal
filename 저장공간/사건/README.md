# 사건 저장공간

## 폴더 구조
사건마다 별도 폴더로 분리한다.

```
사건/
├── 사건명_YYYYMM/
│   ├── 기본정보.md       ← 당사자, 사건번호, 법원, 청구취지
│   ├── 타임라인.md       ← 날짜순 사건 정리
│   ├── 판결문.md         ← 확정사실 / 다투는사실 분리
│   ├── 증거목록.md       ← 증거번호, 종류, 강도 점수
│   ├── 항소이유서.md
│   ├── 답변서.md
│   └── 진행상황.md       ← 현재 단계, 다음 기일
```

## 저장 규칙
- 사건끼리 정보 절대 혼용 금지
- 사용자 승인 후 저장
- 판결문: 확정 사실 / 다투는 사실 반드시 분리
- 감정·추측은 이 공간에 저장 금지 → 임시 공간으로

## 현재 등록된 사건
(없음 — 새 사건 발생 시 하위 폴더 생성)
