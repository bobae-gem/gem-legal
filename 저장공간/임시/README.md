# 임시 저장공간

## 저장 가능한 것
- 감정 메모 ("억울하다", "화가 난다" 등 현재 감정 상태)
- 임시 추측 (증거 없이 추론한 내용)
- 테스트 분석 (확인 전 가설)
- 확정되지 않은 의견
- 실시간 반응

## 핵심 규칙

### 자동 장기저장 금지
이 폴더의 내용은 사용자가 명시적으로 승인하기 전까지
사건 저장공간으로 이동하지 않는다.

### 감정은 사실이 아니다
"상대방이 나쁜 사람 같다" → 임시 공간
"상대방이 계약을 위반했다" (증거 있음) → 사건 공간

### 이동 조건
임시 → 사건 공간 이동:
- 사용자가 "이 내용 사건에 저장해줘"라고 명시한 경우만
