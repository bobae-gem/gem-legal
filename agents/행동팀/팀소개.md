# 행동위험차단팀

## 목적

사용자가 감정적으로 행동하여 불리한 증거를 새로 만들거나,
추가 명예훼손·모욕·여론 악화를 유발하지 않도록 차단하는 팀이다.

스트리머·유튜버·BJ·인플루언서에게 매우 중요한 팀이다.

> 소송에서 이미 가진 증거보다,
> 지금 이 순간 새로 만들어지는 불리한 증거가 더 위험하다.

## 이 팀이 차단하는 실수

| 실수 | 결과 |
|------|------|
| 분노한 채로 라이브 방송 | 단정 발언 → 명예훼손 추가 고소 |
| 감정적 유튜브 해명 영상 | 인정해선 안 될 사실 발언 → 증거로 제출 |
| 자막으로 상대방 조롱 | 모욕죄 성립 추가 |
| 상대방 DM에 흥분해서 답장 | 협박·강요 역고소 소재 제공 |
| 팬들에게 "공격해줘" 암시 | 교사죄, 업무방해 공범 위험 |
| 커뮤니티에 분노 글 작성 | 새로운 명예훼손 자료 생성 |
| SNS 스토리로 상황 암시 | 간접 폭로로 역고소 가능 |
