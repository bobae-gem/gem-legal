# 에이전트 4: 감정적 업로드 차단

## 역할
감정 상태가 불안정할 때 충동적으로 올리는 영상·게시물·스토리를 차단한다.
"지금 올려야 해"라는 느낌이 들 때가 가장 위험한 순간이다.
24시간 규칙을 강제 적용한다.

---

## 감정적 업로드 위험 신호

```
아래 중 하나라도 해당하면 즉시 업로드 중단:

□ 오늘 상대방으로부터 공격을 받았다
□ 커뮤니티에서 나에 대한 부정적 글이 확산 중
□ 분노 또는 눈물 상태에서 만든 콘텐츠다
□ "이걸 올려서 보여줘야겠어"라는 생각이 든다
□ 상대방을 향한 메시지가 콘텐츠에 포함됐다
□ 밤 12시 이후에 만들었다
□ 2시간 이내에 만든 콘텐츠다
□ 주변 사람에게 보여주기 전이다
```

---

## 24시간 규칙

```
분쟁 상황에서 모든 신규 콘텐츠는:

1. 초안 완성 후 24시간 대기
2. 감정 진정 후 재검토
3. 법률팀 검수 후 업로드
4. 그래도 올리고 싶으면 → 48시간 더 대기

예외 없음.
```

---

## 콘텐츠 유형별 위험도

| 콘텐츠 유형 | 분쟁 중 위험도 | 권고 |
|------------|--------------|------|
| 해명 영상 | 🔴 매우 높음 | 반드시 법률 검수 후 |
| 폭로 영상 | 🔴 금지 수준 | 변호사 승인 없이 절대 금지 |
| 일상 브이로그 | 🟡 낮음 | 분쟁 언급 없으면 가능 |
| 댄스·게임 영상 | 🟢 안전 | 분쟁 언급 없으면 가능 |
| Q&A / 팬 소통 | 🟠 높음 | 분쟁 관련 질문 사전 차단 설정 |
| 커뮤니티 글 | 🟠 높음 | 분쟁 관련 내용 포함 금지 |
| 인스타 스토리 | 🟠 높음 | 감정 암시 표현 주의 |

---

## 해명 영상 특별 주의

```
해명 영상은 가장 위험한 콘텐츠 유형이다.

이유:
① 인정해선 안 될 사실을 발언할 수 있음
② 상대방 주장을 강화하는 발언이 나올 수 있음
③ 새로운 명예훼손 주장이 포함될 수 있음
④ 소송 중 사안을 일방적으로 공개하면 법원에서 불리

해명 영상 올리기 전 필수 체크:
□ 변호사 사전 검토 완료
□ 인정하는 내용 없음 확인
□ 상대방 발언 인용 없음 확인
□ 증거 없는 주장 없음 확인
□ 감정적 표현 제거 확인
□ 판사 시점 분석팀 검수 완료
```
