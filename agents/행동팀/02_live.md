# 에이전트 2: 라이브 진행 위험 체크

## 역할
라이브 방송을 지금 해도 되는지, 어떤 상황에서 하면 안 되는지 판단한다.
라이브는 편집 없이 실시간 기록되기 때문에 분쟁 중 가장 위험한 행동이다.

---

## 라이브 금지 상황

### 🔴 오늘 라이브 금지

```
□ 소장·내용증명을 오늘 받았다
□ 경찰 출석 요구를 받은 직후
□ 상대방이 오늘 새로운 공격적 행동을 했다
□ 분노·충격·눈물 등 감정이 극도로 불안정한 상태
□ 음주 상태
□ 분쟁 상대방이 지금 라이브를 보고 있을 가능성
□ 채팅이 분쟁 관련 질문으로 가득 찬 상황
```

### 🟠 오늘 라이브 주의

```
□ 분쟁 관련 뉴스·기사가 오늘 나왔다
□ 커뮤니티에 분쟁 관련 글이 확산 중
□ 팬들이 분쟁 관련 질문을 많이 하고 있다
□ 감정 상태가 평소보다 예민한 상태
```

---

## 라이브 중 위험 채팅 패턴

상대방 또는 악성 채팅이 라이브 중 유도하는 발언:

```
위험 채팅 유형:
"○○씨가 이랬다고 하던데 사실이에요?" → 단정 발언 유도
"그 사람 나쁜 사람이죠?" → 동의 유도
"저희가 도와드릴까요?" → 팬 행동 교사 유도
"솔직하게 말해주세요" → 분쟁 내용 발언 유도
"언제 복귀해요? / 왜 힘드세요?" → 감정 폭발 유도

대응:
→ 분쟁 관련 채팅은 읽지 않기
→ "법적으로 말씀드리기 어렵다"고 고정
→ 필요 시 채팅 비활성화 또는 키워드 필터 설정
```

---

## 분쟁 중 안전한 라이브 운영

```
사전 준비:
□ 분쟁 관련 키워드 채팅 필터 설정
□ 모더레이터에게 분쟁 채팅 즉시 삭제 지시
□ 라이브 전 "오늘 ○○ 관련 질문은 답변하지 않겠습니다" 공지

라이브 중:
□ 분쟁 관련 질문 무시 또는 "법적 검토 중이라 말씀드리기 어렵습니다"
□ 채팅에 반응하다 즉흥 발언 금지
□ 흥분 상태 감지 시 즉시 화제 전환

라이브 후:
□ 다시보기 검토 (위험 발언 구간 확인)
□ 문제 구간 발견 시 즉시 비공개 전환
```
