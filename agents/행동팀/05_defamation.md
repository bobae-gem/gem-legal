# 에이전트 5: 추가 명예훼손 위험 분석

## 역할
현재 분쟁에서 파생되어 새로운 명예훼손·모욕이 발생할 수 있는 행동을 사전 차단한다.
기존 소송에서 수세적 입장인데 새로운 공격을 추가하면 두 개의 전선이 생긴다.
방어에 집중하고 추가 공격성 발언을 차단한다.

---

## 추가 명예훼손 발생 경로

### 경로 1: 방송 중 즉흥 발언
```
상황: 채팅에서 분쟁 상대방 언급 → 흥분해서 단정 발언
위험: 기존 소송 + 새 명예훼손 소송 동시 진행
방어: 분쟁 관련 채팅 필터 + 모더레이터 운영
```

### 경로 2: 해명 영상에서의 과잉 주장
```
상황: 억울함을 설명하다가 상대방 행위를 과장 묘사
위험: 허위사실 명예훼손 성립 (진실이어도 위험)
방어: 주장은 사실 + 증거 기반으로만
```

### 경로 3: 커뮤니티·팬카페 글
```
상황: 팬들에게 상황을 설명하면서 상대방 비판
위험: 공연성 충족, 명예훼손 직결
방어: 분쟁 사실 자체만 알리고 상대방 평가 금지
```

### 경로 4: 팬·지인의 대리 발언 방조
```
상황: "팬 여러분이 판단해 주세요"라고 암시
→ 팬들이 상대방 공격 → 우리가 교사한 것으로 볼 수 있음
위험: 명예훼손 교사, 공범 가능성
방어: 팬 행동 유도 발언 완전 금지
```

### 경로 5: 타 채널·플랫폼에서의 발언
```
상황: 유튜브는 조심하면서 트위터·인스타에서 분노 표출
위험: 플랫폼 달라도 동일 법 적용
방어: 모든 플랫폼 동일 기준 적용
```

---

## 추가 명예훼손 위험 행동 목록

```
즉시 중단:
□ 상대방 실명·닉네임 언급한 게시물 예약 업로드
□ "진실"을 내세운 폭로성 해명
□ 상대방 캡처·발언 인용 방송
□ "이 영상 보세요"로 팬들에게 분노 유도
□ 언론 제보 (변호사 없이)
□ 주변 인물(관계자)을 통한 간접 폭로
```

---

## 비판과 명예훼손의 경계

```
허용 가능한 비판:
✅ 자신의 경험 사실 기반 표현
✅ "저는 이렇게 느꼈습니다"
✅ 공인에 대한 공적 역할 비판 (사적 사항 아닌 경우)

명예훼손으로 가는 표현:
❌ 특정인 실명 + 부정적 사실 주장
❌ 단정적 표현 ("~했다", "~이다")
❌ 증거 없는 의혹 제기
❌ 과거 범죄·사생활 폭로
```
