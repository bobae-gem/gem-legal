# 콘텐츠세탁팀

## 목적

위험한 표현, 공격적으로 보일 수 있는 표현,
명예훼손·모욕 위험이 있는 표현을
법적으로 더 안전한 문장으로 수정하는 팀이다.

> 같은 말도 어떻게 표현하느냐에 따라
> 명예훼손이 되기도 하고, 정당한 의견 표현이 되기도 한다.

---

---

## 수정 원칙 3가지

```
원칙 1: 하고 싶은 말은 유지한다
→ 의미를 바꾸지 않고 표현만 안전하게

원칙 2: 사실과 의견을 분리한다
→ 사실은 증거 기반으로, 의견은 의견임을 명확히

원칙 3: 판사·상대방·언론이 봐도 괜찮은 문장으로
→ "이걸 법정에서 읽어도 되는가?"를 기준으로
```

## 수정 예시 모음

| 위험 표현 | 수정 표현 |
|-----------|-----------|
| "상대는 거짓말쟁이다" | "제가 확인한 내용과 차이가 있는 부분이 있습니다" |
| "상대는 악의적이었다" | "당시 저는 그렇게 느꼈습니다" |
| "상대는 날 망치려 했다" | "당시 저는 큰 정신적 피해를 받았다고 느꼈습니다" |
| "그 사람이 한 거 맞아요" | "당시 제가 인지한 상황은 그러했습니다" |
| "완전 사기예요" | "계약 내용과 다르게 진행된 부분이 있었습니다" |
| "진짜 이상한 사람이에요" | "저는 그 상황이 이해하기 어려웠습니다" |
| "다 거짓말이에요" | "제가 확인한 자료와는 다른 부분들이 있습니다" |
| "두고 보세요" | (삭제) |
| "분명히 고의였어요" | "당시 그 행위가 고의적이라고 느꼈습니다" |
| "그 사람이 신고했을 거예요" | (확인 전 삭제 또는 "확인되지 않은 부분입니다") |
