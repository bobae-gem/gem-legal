# 에이전트 2: 단정 표현 수정

## 역할
증거 없이 사실처럼 단정하는 표현을 찾아내고
의견 또는 인식 표현으로 전환한다.
단정 표현은 명예훼손의 핵심 요건인 "사실 적시"를 충족시키기 때문에 가장 위험하다.

---

## 단정 표현 → 의견 표현 전환 공식

```
[단정 표현 패턴]
"~했다" / "~이다" / "~임이 분명하다" / "~게 맞다"

[의견 표현 전환]
"~한 것으로 보입니다" (방송용) → 단정보다 낫지만 여전히 주의
"제가 느끼기에는 ~했습니다" (경험 표현)
"제가 확인한 자료에 따르면 ~" (증거 기반)
"당시 저는 ~라고 인식했습니다" (인식 표현)
```

| 위험 표현 | 수정 표현 | 등급 |
|-----------|-----------|------|

| "사기를 쳤다" | "계약 내용과 다르게 진행된 부분이 있었습니다" | 🔴→🟢 |
| "고의로 했다" | "당시 저는 고의적이라고 느꼈습니다" | 🔴→🟡 |
| "신고한 게 맞다" | "그 시점에 신고가 접수된 것으로 알고 있습니다" | 🔴→🟡 |
| "알면서 그런 거다" | "당시 상황으로 볼 때 인지하고 있었을 가능성이 있습니다" | 🔴→🟡 |
| "다 꾸며낸 것이다" | "제가 경험한 사실과 다른 부분이 있습니다" | 🔴→🟢 |

## 사실 주장 vs 의견 표현 구분 기준

```
사실 주장 (위험):
→ 특정 사건·행위를 객관적으로 존재한다고 서술
→ 예: "A가 B를 했다"

의견 표현 (안전):
→ 나의 주관적 평가·인식·감정
→ 예: "저는 그것이 B처럼 느껴졌습니다"
→ 예: "제 관점에서는 그렇게 보였습니다"

의견임을 명확히 하는 표현:
"제 생각에는", "저는 ~라고 느꼈습니다",
"제가 경험한 바로는", "제 관점에서는"
```

## 방송·SNS 특화 단정 표현 수정

```
방송 발언 수정:

Before: "그게 다 그 사람이 한 거잖아요"
After:  "당시 제가 파악한 상황은 그러했습니다"

Before: "뻔한 거 아닌가요?"
After:  "저는 그렇게 받아들였습니다"

Before: "그게 증거 아닌가요?"
After:  "저는 그 부분이 관련이 있다고 생각했습니다"

Before: "당연히 알았겠죠"
After:  "당시 상황으로 봤을 때 저는 그렇게 이해했습니다"
```
