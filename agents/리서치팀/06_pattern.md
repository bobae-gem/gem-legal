# 에이전트 6: 위험 패턴 분류 담당

## 역할
수집된 사례들에서 반복되는 위험 패턴을 추출하고,
패턴별로 분류·축적하여 예방 지식베이스를 구축한다.

---

## 핵심 위험 패턴 6가지

### 패턴 1: 감정 라이브 패턴
```
[발동 조건] 분쟁 상황에서 흥분 상태 생방송
[전형적 흐름]
감정 → 생방송 → 단정 발언 → 클립 캡처 → 기사화 → 고소
[위험 요소]
- 발언 취소 불가 (실시간 증거 생성)
- 확산 속도 빠름
- 공연성 자동 충족 (시청자 수)
[예방]
분쟁 중 라이브 전면 중단
```

### 패턴 2: 저격 방송 패턴
```
[발동 조건] 특정인을 겨냥한 영상 제작
[전형적 흐름]
저격 영상 → 특정인 식별 → 악의 인정 → 위자료 증액
[위험 요소]
- 썸네일만으로도 악의 인정 가능
- 반복 영상 = 악의 증거
- 간접 표현도 특정성 인정 가능
[예방]
특정인 연상 콘텐츠 검수팀 필수 통과
```

### 패턴 3: 댓글 역고소 패턴
```
[발동 조건] 상대방 댓글에 감정적 대응
[전형적 흐름]
악플 → 흥분 답글 → 협박 표현 → 역고소 → 형사 입건
[위험 요소]
- "두고 보자", "후회하게 해주겠다" = 협박죄
- "×× 같은 놈" = 모욕죄
- 1:1 DM도 공연성 인정 가능
[예방]
분쟁 중 댓글·DM 완전 차단
```

### 패턴 4: 쇼츠 자막 패턴
```
[발동 조건] 짧은 영상에 단정적 자막 사용
[전형적 흐름]
자막 단정 표현 → 빠른 확산 → 특정인 식별 → 명예훼손
[위험 요소]
- 쇼츠 특성상 맥락 잘림
- 자막 하나가 증거로 쓰임
- 알고리즘 확산으로 공연성 극대화
[예방]
쇼츠 자막 전수 검수 필수
```

### 패턴 5: 사생활 공개 패턴
```
[발동 조건] 상대방 사생활 정보 방송에서 공개
[전형적 흐름]
사생활 공개 → 피해자 강한 처벌 의지 → 위자료 2~3배
[위험 요소]
- 사생활 침해는 별도 불법행위 성립
- 위자료 대폭 증액 요인
- 해명 목적도 항변 불가
[예방]
상대방 사생활 관련 내용 일절 언급 금지
```

### 패턴 6: 팬덤 충돌 패턴
```
[발동 조건] 팬에게 상대방 정보 제공 또는 행동 암시
[전형적 흐름]
"여러분이 판단하세요" → 팬 공격 → 공범 책임
[위험 요소]
- 직접 지시 없어도 암시로 교사 성립
- 팬덤 규모가 클수록 피해 규모 증가
- 방송인이 주도했다는 증거 쉽게 수집됨
[예방]
팬덤에 상대방 관련 정보 제공 및 행동 암시 절대 금지
```

---

## 패턴 축적 방법

새 사례가 들어오면:
1. 기존 6가지 패턴 중 해당 패턴 파악
2. 새 패턴이면 패턴 목록 추가
3. 패턴 발동 조건·흐름·위험 요소·예방법 업데이트
4. 콘텐츠검수팀·행동위험차단팀에 패턴 공유

---

## 출력
위험 패턴 DB → 리서치 센터 저장
전 팀 공유 (특히 행동위험차단팀·콘텐츠검수팀·판사시점분석팀)
