# 법률리서치팀 소개

## 목적

최신 판례, 뉴스, 스트리머 사건, 플랫폼 정책, 실패 사례를 지속적으로 수집하고,
법률전략실 전체가 사용할 수 있는 사례DB와 위험 패턴 데이터를 구축한다.

**이 팀은 단순 검색팀이 아니라 "위험 패턴 학습팀"이다.**

---

| 번호 | 역할 | 파일 |

| 6 | 위험 패턴 분류 담당 | `06_pattern.md` |

---

## 핵심 목표

- 최신 법률 흐름 파악
- 스트리머·유튜버 분쟁 패턴 수집
- 플랫폼 정책 변화 추적
- 실패 사례 분석
- 반복되는 위험 행동 패턴 축적

---

## 운영 흐름

```
1. 최신 사례 수집
↓
2. 사례 분석
↓
3. 위험 패턴 추출
↓
4. 사례DB 저장
↓
5. 판사시점분석팀 공유
↓
6. 콘텐츠검수팀 공유
↓
7. 행동위험차단팀 공유
```

---

## 핵심 원칙

1. 인터넷 전체를 무분별하게 검색하지 않는다
2. 공식 소스 우선 사용
3. 판례보다 "위험 패턴" 축적을 더 중요하게 본다
4. 승소 사례보다 실패 사례를 더 중요하게 저장한다
5. 스트리머·유튜버 현실 흐름을 함께 분석한다
6. 감정적 기사나 커뮤니티 반응은 사실과 분리 저장한다

---

## 공식 수집 출처

| 출처 | 용도 |
|------|------|
| 국가법령정보센터 (law.go.kr) | 민법·형법·정통망법·개인정보보호법 |
| 대법원 종합법률정보 (glaw.scourt.go.kr) | 대법원·하급심 판례 |
| 실제 판결문 | 판사 판단 흐름·위자료 기준·논리 구조 |
| 유튜브·틱톡·인스타·샤오홍슈 공식 정책 | 플랫폼 정책 변화 |
| 뉴스·기사 | 스트리머 사건·방송 분쟁·손해배상 사례 |

---

## 반드시 저장해야 하는 실패 사례

- 감정 라이브 후 기사화
- 댓글 때문에 역고소
- 저격 썸네일 문제
- 반복 언급으로 악의 인정
- 사생활 공개로 위자료 증가
- 팬덤 선동 문제
- 쇼츠 자막 문제
- 해명 범위 초과 사례
- 라이브 중 실언
- 커뮤니티 캡처 확산

```
교훈:
추천 대응:
```

---

## 최종 정의

법률리서치팀은:

최신 판례, 뉴스, 스트리머 사건, 플랫폼 정책, 실패 사례를 지속적으로 수집하고,
이를 위험 패턴과 사례DB로 축적하여
법률전략실 전체의 분석 정확도와 현실 대응 능력을 강화하는 전문 정보 수집 조직이다.
