# 증거강도분석팀

## 목적

증거의 신뢰도, 직접성, 편집 가능성, 설득력을 분석하여
재판에서 얼마나 강하게 사용 가능한지 평가하는 팀이다.

> 좋은 증거를 약하게 쓰는 것보다,
> 약한 증거를 강하게 쓰는 것이 더 위험하다.

## 분석 대상 자료

| 자료 유형 | 주요 위험 요소 |
|-----------|--------------|
| 카카오톡 캡처 | 대화 누락, 원본 미보유, 맥락 왜곡 |
| 문자 메시지 | 발신자 특정, 시간 조작 의혹 |
| 녹취·음성 파일 | 편집 의혹, 맥락 누락, 동의 여부 |
| 영상 | 편집 의혹, 원본 파일 보유 여부 |
| CCTV | 보관 기한, 원본 제출 경로 |
| 언론 기사 | 사실 여부, 취재원, 기사 신뢰도 |
| SNS 캡처 | 삭제 전 공증 여부, URL 포함 여부 |
| 이메일 | 헤더 포함 여부, 수신 확인 |
| 통화기록 | 통화 내용 미확인, 존재 입증만 가능 |
| 전자소송 자료 | 제출 기한, 파일 무결성 |

## 증거 강도 5단계

| 등급 | 점수 | 의미 |
|------|------|------|
| S급 | 90~100 | 재판부가 그대로 채택 가능 |
| A급 | 75~89 | 강한 증거, 보완 시 완성 |
| B급 | 55~74 | 보조 증거로 활용 가능 |
| C급 | 35~54 | 단독 사용 위험, 다른 증거와 결합 필요 |
| D급 | 0~34 | 사용 신중, 역효과 위험 |
