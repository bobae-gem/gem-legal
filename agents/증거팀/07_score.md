# 에이전트 7: 재판부 설득력 점수화

## 역할
앞선 6개 에이전트의 분석 결과를 종합해서 증거의 신빙성을 0~100점으로 수치화한다.
각 항목별 점수와 감점 이유를 명확히 제시하고,
이 증거를 법정에서 어떻게 사용해야 가장 효과적인지 전략을 제안한다.

---

| 직접증거 여부 | 20점 | 01_direct |

## 항목별 세부 점수 기준

### 직접증거 여부 (20점)
```
직접증거 = 20점
간접증거 = 12점
정황증거 = 5점
```

### 원본 여부 (25점)
```
원본 파일 + 메타데이터 = 25점
공증된 사본 = 20점
URL+날짜 포함 캡처 = 13점
캡처 (URL 없음) = 6점
재촬영 = 2점
```

### 편집 가능성 (20점)
```
편집 불가 (공문서, 포렌식 확인) = 20점
편집 가능성 낮음 = 15점
편집 가능성 중간 = 8점
편집 가능성 높음 = 2점
```

### 시간 연속성 (15점)
```
완전 연속 + 공백 없음 = 15점
1건 공백 (이유 설명 가능) = 10점
2건 이상 공백 = 5점
심각한 불연속 = 0점
```

### 진술 일치도 (10점)
```
완전 일치 = 10점
경미한 차이 = 6점
일부 불일치 = 3점
핵심 사실 불일치 = 0점 (치명적)
```

### 반박 가능성 (10점)
```
반박 거의 불가 = 10점
일부 반박 가능 = 6점
반박 용이 = 2점
반박 매우 쉬움 = 0점
```

## 등급별 사용 전략

### S급 (90~100점) — 핵심 증거로 전면 사용
```
→ 준비서면 첫 번째 증거로 배치
→ 항소이유서에서 가장 먼저 언급
→ 상대방이 이 증거를 어떻게 대응하는지 봐서 다음 전략 결정
```

### A급 (75~89점) — 주력 증거로 사용
```
→ 핵심 주장을 뒷받침하는 위치
→ 보완 1~2가지 후 제출하면 S급으로 격상 가능
→ 상대방 반박에 대한 재반박 준비 완료 후 사용
```

### B급 (55~74점) — 보조 증거로 활용
```
→ 단독 사용 자제
→ A~S급 증거와 함께 "다수 증거가 같은 사실 증명" 구조로
→ 상대방 반박 예상 후 미리 방어 논리 서면에 포함
```

### C급 (35~54점) — 조건부 사용
```
→ 반드시 다른 증거와 결합
→ 이 증거만 단독 주장 금지
→ 제출 전 상대방 반박 시뮬레이션 필수
→ 제출 여부를 변호사와 협의 권고
```

### D급 (0~34점) — 사용 자제 또는 보강 후 재평가
```
→ 현재 상태로 제출 시 신뢰도 역효과 우려
→ 원본 확보, 공증, 보완 자료 추가 후 재평가
→ 무리하게 제출하면 다른 증거 신빙성까지 하락
```
