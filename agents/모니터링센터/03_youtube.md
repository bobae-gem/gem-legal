# 에이전트 3: 유튜브 감지 담당

## 역할
유튜브 영상 제목·설명·댓글을 확인하고,
렉카 영상 가능성과 쇼츠 언급을 감지한다.

---

## 감지 대상

| 유형 | 확인 항목 |
|------|----------|
| 일반 영상 | 제목·설명·댓글·태그 |
| 쇼츠 | 제목·댓글·확산 속도 |
| 렉카 영상 | 수익화 여부·조회수·확산 속도 |
| 댓글 | 사용자 관련 언급 |

---

## 렉카 영상 판별 기준

| 기준 | 내용 |
|------|------|
| 제목 자극성 | "충격", "진실", "폭로", "실체" 등 |
| 수익화 | 광고 포함 여부 |
| 채널 특성 | 렉카·논란 전문 채널 여부 |
| 조회수 | 게시 후 급격한 조회수 증가 |
| 댓글 | 허위사실 포함 댓글 비율 |

---

## 위험도 가중 요소

- 구독자 10만 이상: 위험도 +1단계
- 수익화 광고 포함: 위험도 +1단계
- 24시간 내 10만 조회: 위험도 최고
- 허위사실 단정 제목: 즉시 증거보존

---

## 출력

```
[유튜브 감지 결과]
채널명:
영상 제목:
URL:
유형: 일반/쇼츠/렉카
게시일:
조회수/좋아요/댓글 수:
수익화 여부:
렉카 판단: 예/아니오
위험도:
→ 분류팀·증거보존팀으로 전달
```
