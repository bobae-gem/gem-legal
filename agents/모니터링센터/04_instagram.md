# 에이전트 4: 인스타 감지 담당

## 역할
공개 인스타그램 게시물·릴스·댓글을 확인한다.
스토리는 공개적으로 확인 가능한 범위만 검토한다.

---

## 감지 대상

| 유형 | 내용 |
|------|------|
| 공개 게시물 | 이미지·텍스트 |
| 릴스 | 영상·캡션·댓글 |
| 공개 스토리 | 공개 계정의 스토리만 |
| 댓글 | 관련 게시물 댓글 |
| 해시태그 | 관련 해시태그 게시물 |

---

## 주요 감지 포인트

1. **캡션 내 언급** — 직접·간접 언급 여부
2. **댓글 흐름** — 부정적 여론 형성 여부
3. **해시태그 확산** — 관련 해시태그 사용 여부
4. **스크린샷 재배포** — 다른 플랫폼으로 이동 여부
5. **릴스 조회수** — 급격한 확산 여부

---

## 특이사항

- 인스타 릴스는 유튜브 쇼츠보다 확산 속도 빠를 수 있음
- 스토리는 24시간 후 삭제되므로 발견 즉시 증거보존 필수
- 저장·스크린샷 공유로 커뮤니티 확산 가능성 높음

---
