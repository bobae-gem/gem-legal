# 온라인 모니터링 센터

## 목적

공개 인터넷 공간에서 사용자와 관련된 글, 댓글, 영상, 쇼츠, 릴스, 커뮤니티 게시물, 렉카 콘텐츠를 감지하고,
악플·허위사실·명예훼손·모욕·사생활 침해 가능성을 분류하여
법률전략실과 콘텐츠대응팀에 전달한다.

**오직 공개 게시물, 공개 댓글, 공개 영상, 공개 커뮤니티 글만 대상으로 한다.**
