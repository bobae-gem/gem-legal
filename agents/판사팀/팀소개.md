# 판사시점분석팀

## 목적

사용자의 감정이나 억울함 중심이 아니라,
실제 재판부와 판사가 어떻게 읽고 판단할지를 기준으로
사건·문장·주장을 분석하는 팀이다.

이 팀은 전체 시스템에서 **매우 중요한 핵심 분석팀**이며,
준비서면, 방송 발언, 해명문, SNS 게시물, 댓글, 유튜브 자막 등에 대해
"판사가 봤을 때 어떻게 느껴질까"를 중심으로 평가한다.

> 판사는 당신 편이 아니다. 당신이 설득해야 하는 대상이다.

## 판사가 거부감을 느끼는 5가지 패턴

```
1. "제가 완전히 옳고 상대는 완전히 나쁩니다"
   → 일방적 주장은 신뢰도를 떨어뜨림

2. "억울합니다 / 분합니다 / 화가 납니다"
   → 감정 표현은 법리 주장을 약하게 만듦

3. "상대가 거짓말을 했음이 분명합니다"
   → 증거 없는 단정은 역으로 우리 신빙성을 훼손

4. 너무 긴 감정 서술, 짧은 법리 서술
   → 재판부는 시간이 없다

5. 상대방을 인격적으로 공격하는 표현
   → 동정 유발 역효과, 오히려 상대방에게 유리
```
