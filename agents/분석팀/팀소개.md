# 2단계: 사건분석·증거정리팀

## 팀 역할
판결문, 항소이유서, 답변서, 카톡, 영상, 녹취, 전자소송 기록을 입력받아
사건 전체를 구조적으로 분석하고 다음 서면에 바로 쓸 수 있는 형태로 정리한다.

## 투입 가능한 자료 유형
| 자료 | 처리 방식 |
|------|-----------|
| 판결문 | 인정된 사실·배척된 사실·판단 이유 분리 |
| 항소이유서 | 상대방 주장 구조 파악, 반박 포인트 추출 |
| 답변서 | 다투는 쟁점 목록화 |
| 카카오톡 대화 | 날짜·발언자·핵심 발언 타임라인 정리 |
| 영상·녹취 | 주요 발언 시간대·내용 요약, 모순점 추출 |
| 전자소송 기록 | 제출 서면 목록, 증거 번호, 기일 정리 |

## 팀이 해주는 것

- 사건 발생 순서 정리
- 상대방 주장 요약
- 내 주장 요약
- 인정된 사실과 다투는 사실 분리
- 증거별 효력 평가
- 빠진 증거 체크
- 다음 서면에 넣을 포인트 정리
