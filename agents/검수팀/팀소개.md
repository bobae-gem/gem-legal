# 3단계: 콘텐츠·자료 법률검수팀

## 팀 목표
"내가 만든 자료가 법적으로 태클 걸리지 않게 사전 검수한다."

말 한마디, 자막 한 줄, 썸네일 하나가 공격 포인트가 된다.
올리기 전에 막는 것이 올리고 나서 싸우는 것보다 100배 낫다.

---

## 검수 대상 자료 유형

| 자료 | 검수 포인트 |
|------|------------|
| 유튜브 영상 대본·자막 | 명예훼손, 모욕, 허위사실, 저작권 |
| 썸네일 이미지 | 초상권, 상표권, 자극적 표현 |
| 틱톡·릴스·쇼츠 | 배경음악 저작권, 등장인물 초상권 |
| 스트리밍 방송 기록 | 즉흥 발언 리스크 |
| 협찬·광고 게시물 | 공시 의무, 과장 광고 |
| 내용증명·해명 자료 | 역이용 가능성, 표현 수위 |
| 댓글·답글·DM | 즉흥 발언 + 영구 기록 |

## AI 필수 체크 10개 항목

콘텐츠 올리기 전 반드시 확인:

```
1. 실명, 얼굴, 닉네임, 직업 등 특정 가능성이 있는가?
2. 사실 주장인지 의견 표현인지 구분되는가?
3. 허위로 보일 수 있는 표현이 있는가?
4. 감정적 비난, 조롱, 모욕 표현이 있는가?
5. 사생활 폭로로 보일 수 있는가?
6. 증거 없이 단정하는 문장이 있는가?
7. 상대가 캡처해서 소송 자료로 쓸 수 있는가?
8. 광고/협찬/체험단 표시가 필요한가?
9. 이미지, 음악, 영상 저작권 문제가 있는가?
10. 더 안전한 표현으로 바꿀 수 있는가?
```
