# 자동반영 금지 규칙

> 보배 법률전략실 AI 시스템 운영 원칙
> 이 파일에 명시된 내용은 dashboard_data.json(확정 데이터)에 절대 자동 반영하지 않는다.
> 반드시 사용자(대표) 승인 후 approved 상태로만 반영한다.

---

## 자동반영 절대 금지 항목

### 1. 감정 표현
사용자의 감정 상태, 분노, 억울함, 두려움 등 주관적 감정 표현

예시:
- "억울해요", "화가 나요", "무서워요"
- "상대방이 너무해요"
- "이건 말이 안 돼요"

처리 방법: 임시 메모(memo) 카테고리로만 기록

---

### 2. 추측성 표현
확인되지 않은 가정, 가능성, 예상에 기반한 주장

예시:
- "아마 ~일 것 같아요"
- "~일 수도 있어요"
- "~인 것 같아요"
- "~했을 거예요"

처리 방법: 검토 대기(review) 카테고리로 기록, 사용자 확인 후 반영

---

### 3. 상대방에 대한 단정적 비난
증거 없이 상대방의 의도·성격·행위를 단정하는 표현

예시:
- "상대방이 일부러 했다" → 사용자 주장(user_claim)이지 확정 사실(confirmed) 아님
- "악의적으로 행동했다" → 법원이 인정해야 확정
- "거짓말을 했다" → 증거·판결 기반이어야 확정

처리 방법: 사용자 주장(user_claim) 카테고리로만 기록

---

### 4. 미확인 사실
공식 문서·판결문·전자소송 기록으로 확인되지 않은 사실관계

예시:
- 기억에 의존한 날짜·금액
- 상대방 말을 전달한 내용
- 언론 보도만 근거로 한 사실

처리 방법: 검토 대기(review)로 기록, 문서 확인 후 confirmed으로 이동

---

### 5. 법률 결과 예측
소송 결과, 판결 내용, 인용 금액 등에 대한 예측

예시:
- "이기면 얼마 받을 것 같아요?"
- "항소심에서 뒤집힐 것 같아요"
- "위자료가 올라갈 것 같아요"

처리 방법: 임시 메모(memo)로만 기록, 절대 분석 결과(analysis)로 반영 금지

---

### 6. 위험도 수치 자동 변경
AI가 위험도 % 또는 등급을 임의로 변경하는 것

예시:
- "위험도를 높음으로 자동 변경"
- 사건 발생으로 위험도 수치 자동 업데이트

처리 방법: 위험도 변화 제안(risk)으로만 기록, 사용자 승인 후 변경

---

### 7. 전략 아이디어 (AI 제안 수준)
AI가 제안하는 소송 전략, 대응 방향 등

예시:
- "준비서면에 이 내용을 추가하면 좋겠어요"
- "이 판례를 활용하는 게 유리할 것 같아요"

처리 방법: pending으로 기록, 대표 검토 후 approved 처리

---

## 확정 가능 항목 (approved 반영 가능)

| 항목 | 조건 |
|------|------|
| 확정 사실 | 판결문·전자소송·공식 서면 기반 |
| 공식 문서 | 법원·기관 발행 문서 원문 기반 |
| 증거 | 직접 제출된 자료 기반 |
| 긴급 알림 | 전자소송 확인·공식 통보 기반 |
| 분석 결과 | AI 신뢰도 70% 이상 + 출처 명확 |
| 추천 행동 | 사실관계 확정된 상황 기반 |

---

## AI 신뢰도 기준

| 신뢰도 | 설명 | 반영 가능 여부 |
|--------|------|---------------|
| 90% 이상 | 공식 문서 직접 기반 | 승인 후 반영 가능 |
| 70~89% | 복수 근거 있음 | 승인 후 반영 가능 |
| 50~69% | 단일 근거 또는 추론 | 검토 대기 필요 |
| 50% 미만 | 추측 또는 불충분한 근거 | 임시 메모만 가능 |

---

## 상태(Status) 체계

| 상태 | 의미 | 표시 위치 |
|------|------|-----------|
| pending | AI 제안, 대표 검토 대기 | 상황실 업데이터 |
| approved | 대표 승인 완료 | 메인 상황실 |
| hold | 보류 (추후 재검토) | 상황실 업데이터 |
| rejected | 삭제됨 | 변경 기록만 |

---

## 출처(Source) 필수 표기 기준

모든 pending 항목은 출처를 반드시 명시해야 한다.

허용 출처:
- 판결문 (사건번호 포함)
- 전자소송 기록
- 사용자 대화
- 답변서·항소이유서 등 서면
- 콘텐츠 검수실 분석
- AI 팀 분석 (팀명 명시)

금지 출처:
- 불명확한 "인터넷 자료"
- 출처 없는 법률 정보
- AI가 자체 생성한 사실
