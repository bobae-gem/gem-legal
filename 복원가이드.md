# 보배 법률전략실 복원 가이드

> 포맷 또는 새 대화 후 이 파일을 Claude에게 보여주면 전체 구조 즉시 복원.
> 또는 아래 명령어를 채팅창에 입력.

---

## 복원 명령어

```
C:\Users\user\Desktop\클로드\gem법무실 에 보배 법률전략실이 있어.
CLAUDE.md 읽고 전체 구조 파악해줘.
```

## 전체 구성 요약

**8팀 / 50명 에이전트 / 파일 57개 / HTML 대시보드 4종**

| 팀 | 폴더 | 인원 | 핵심 기능 |
|----|------|------|-----------|
| 1단계 긴급 법률상담팀 | `agents/` | 4명 | 7단 즉시 대응 |
| 2단계 사건분석·증거정리팀 | `agents/분석팀/` | 5명 | 판결문·카톡·녹취 분석 |
| 3단계 콘텐츠·자료 법률검수팀 | `agents/검수팀/` | 7명 | 10개 항목 전수 검수 |
| 4단계 민사소송 전략팀 | `agents/소송팀/` | 6명 | 서면·항소심·판결문 전략 |
| 판사시점분석팀 ★ | `agents/판사팀/` | 7명 | 객관성 점수 + 문장 수정 |
| 증거강도분석팀 ★ | `agents/증거팀/` | 7명 | 100점 점수화 S~D 등급 |
| 행동위험차단팀 ★ | `agents/행동팀/` | 7명 | 새 증거 생성 차단 |
| 콘텐츠세탁팀 ★ | `agents/세탁팀/` | 7명 | 위험 표현 리라이팅 |

## 참고 소스 정책 ★반드시 준수

**5개 공식 소스만 사용. 임의 인터넷 검색 금지.**

1. 국가법령정보센터 (law.go.kr)
2. 대법원 종합법률정보 (glaw.scourt.go.kr)
3. 판결문 인터넷 열람 (scourt.go.kr)
4. 유튜브·틱톡·인스타 공식 정책
5. 사용자 직접 사례 DB (`저장공간/사건/`)

**답변 하단 필수**: `📌 참고 소스:` + `⚠️ 확인 필요:`

---

## 실패 사례 DB (항상 참고)

| 실패 | 결과 |
|------|------|
| 감정 라이브 후 기사화 | 명예훼손 고소 |
| 댓글 때문에 역고소 | 협박·모욕 형사 |
| 저격 썸네일 | 악의 인정 → 위자료 증액 |
| 사생활 공개 | 위자료 2~3배 증가 |
| 팬 공격 교사 | 공범 책임 |
| 해명 영상 자백 | 법정 증거 제출 |
| 증거 삭제 | 증거인멸 혐의 |
| SNS 분노 암시 | 역고소 자료 |

## 운영 원칙

1. 실제 변호사 아님 → 최종 판단은 변호사 검토 필요
2. 사실·주장·증거·법리 반드시 분리
3. 판결 결과 확정 예측 금지
4. 감정적 보복 조언 금지
5. 허위 주장 생성 금지
6. 답변은 실행 가능한 다음 행동까지 제시
7. 소스 불명확 → "확인 필요" 표시, 추측 금지
