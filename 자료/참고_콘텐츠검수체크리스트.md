# 참고자료: 콘텐츠 법률검수 체크리스트
> 구 3단계 콘텐츠검수팀 자료. 영상·게시물 올리기 전 검수 참고용.

## 공통 체크리스트 (모든 콘텐츠)

```
□ 타인의 저작물 사용 시 라이선스 확인
□ 등장인물 동의 여부 (초상권)
□ 미성년자 등장 시 보호자 동의 + 아동복지법 검토
□ 개인정보 노출 여부 (얼굴, 주소, 전화번호, 차량번호)
□ 특정인 명예훼손·모욕 가능성
□ 허위사실 포함 여부
□ 광고·협찬 표시 의무 이행
□ 의료·금융·법률 정보 제공 시 면책 문구
□ 혐오표현·차별 금지
□ 개인 간 분쟁 폭로성 콘텐츠 여부
```

## 채널별 추가 체크리스트

### gemdance (댄스 커버)
```
□ 음악 저작권: 유튜브 Content ID 허용 여부 확인
□ 안무 저작권: 원안무가 커버 허용 여부 확인
□ 의상·소품 상표권 클로즈업 주의
□ 장소 촬영 허가 (실내 공연장, 상업시설)
□ 원곡 아티스트 이미지 훼손 가능성
```

### gemmama (육아 콘텐츠)
```
□ 자녀 얼굴 노출 수위 (아동 개인정보 보호)
□ 아동 프라이버시 침해 가능성
□ 육아용품 협찬 시 효능 과장 금지
□ 의료·건강 정보 면책 문구
□ 다른 아동 등장 시 보호자 동의
□ 아동학대처벌법 저촉 가능성 (훈육 영상 등)
```

### gemtour / gemkorea (여행 콘텐츠)
```
□ 촬영 금지 구역 여부 (군사시설, 문화재, 사유지)
□ 현지인 동의 없는 얼굴 노출
□ 해외 현지법 저촉 여부 (국가별 촬영 규제)
□ 드론 촬영 허가
□ 상업시설 내부 촬영 허가
□ 특정 시설·사업체 부정적 묘사
```

### gemAI (AI 콘텐츠)
```
□ AI 생성물임을 명시 (허위 인간 창작물 주장 금지)
□ 실존 인물 AI 생성 이미지 동의 여부
□ AI 딥페이크 오해 방지 문구
□ 저작권 학습 데이터 출처 문제 (플랫폼 정책 확인)
□ AI 음성 합성 시 원 성우·가수 동의
```

### gemmorning (일상 콘텐츠)
```
□ 지인·가족 동의 없는 노출
□ 일상에서 촬영된 타인 얼굴 처리
□ 특정 장소·브랜드 부정적 언급
□ 개인 의견을 사실처럼 표현한 부분
```

## 협찬·광고 공시 의무

| 구분 | 올바른 표시 | 잘못된 표시 |
|------|------------|------------|
| 유료 협찬 | "#유료광고", "#협찬", "#AD" | 해시태그 끝에 숨기기 |
| 무료 제공 | "#무료제공", "#협찬제품" | 아예 미표시 |
| 제휴 마케팅 | "#파트너십", "#제휴마케팅" | 링크만 있고 표시 없음 |

**표시 위치**: 본문 첫 부분 또는 잘 보이는 위치 (접기 전)

**공시 위반 시 제재**: 공정위 과태료 최대 500만원, 유튜브 수익화 정지

## 협찬 계약서 필수 확인 항목

```
□ 콘텐츠 수정·삭제 권한 (브랜드에게 있으면 위험)
□ 독점 조항 (경쟁사 협찬 금지 기간)
□ 위약금 산정 기준 (조회수·구독자 연동 시 주의)
□ 저작권 귀속 (창작자 vs 브랜드)
□ 2차 사용 허락 범위 (브랜드의 광고 재활용)
□ 계약 해지 조건 및 절차
□ 분쟁해결 방법 (중재 vs 소송, 관할법원)
□ 정산 기준일 및 지연이자
```
