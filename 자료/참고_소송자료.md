# 참고자료: 민사소송 자료
> 구 4단계 민사소송 전략팀 자료. 준비서면 양식, 위자료 기준, 항소심 체크리스트 참고용.

## 청구원인 유형

```
□ 불법행위로 인한 손해배상 (민법 750조)
□ 채무불이행으로 인한 손해배상 (민법 390조)
□ 저작권 침해 손해배상 (저작권법 125조)
□ 명예훼손 손해배상
□ 계약위반 위약금 청구
□ 부당이득반환 청구
```

### 청구취지 기본 문구
```
피고는 원고에게 금 ___원 및 이에 대하여 ___부터 이 사건
소장 송달일까지는 연 5%, 그 다음날부터 다 갚는 날까지는
연 12%의 각 비율에 의한 금원을 지급하라.
```

## 요건사실

**불법행위 청구 4가지 요건:**
1. 고의 또는 과실
2. 위법성
3. 손해 발생
4. 인과관계

**채무불이행 청구 요건:**
1. 유효한 계약의 존재
2. 채무의 내용
3. 채무불이행 사실
4. 귀책사유
5. 손해 발생 및 인과관계

## 항소심 준비 체크리스트

```
□ 1심 판결문 전체 분석 (법원이 인정한 사실 vs 배척한 사실)
□ 1심에서 제출 못한 증거 목록 작성
□ 1심 패소 이유 분석 (무엇이 부족했는가)
□ 항소이유서 제출 기한 확인 (항소장 제출 후 20일)
□ 새 증인 신청 여부 검토
□ 법리 보강 (유사 판례 추가 수집)
```

### 항소 이유 유형

| 유형 | 내용 | 성공률 |
|------|------|--------|
| 사실오인 | 1심이 증거를 잘못 판단 | 중간 |
| 법리오해 | 1심이 법을 잘못 적용 | 높음 |
| 채증법칙 위반 | 증거 평가 방식의 오류 | 낮음 |
| 새로운 증거 | 1심에 없던 증거 제출 | 경우에 따라 높음 |
| 양형부당 (형사) | 형량이 너무 무겁거나 가벼움 | 중간 |

## 합의금 산정 기준

```
합의 하한선 = 승소 예상금액 × 승소 가능성 × 0.7
합의 상한선 = 승소 예상금액 × 0.9
```

### 공탁 제도
- 형사 합의 대신 공탁으로 피해변제 의사 표시
- 공탁 후 피해자가 수령하지 않아도 양형에 유리
- 공탁금 = 피해액 + 지연이자

## 소송 비용 예측

| 항목 | 비용 |
|------|------|
| 인지대 | 청구금액 × 0.0045 (1심 기준) |
| 변호사 선임료 | 통상 300만~1,500만원 |
| 성공보수 | 합의금·판결금의 5~15% |
| 감정 비용 | 필요 시 100만~500만원 |
| 항소심 추가 비용 | 1심의 70~100% 수준 |
