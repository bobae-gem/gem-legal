# 참고자료: 긴급 대응 매뉴얼
> 구 1단계 긴급 법률상담팀 자료. 채널별 긴급상황 대응 참고용.

## 긴급도 분류표

| 분류 | 예시 | 긴급도 |
|------|------|--------|
| 형사 위협 | 고소, 경찰 출석, 고발 | 🔴 최고 |
| 민사 청구 | 내용증명, 손해배상 청구 | 🟠 높음 |

## 상황별 긴급 대응 매뉴얼

### 내용증명 받았을 때
1. **읽기만 하고 당장 답변 금지** — 14일 이내 대응 검토
2. 발송인, 주장 내용, 요구사항, 기한 파악
3. 내용증명 자체는 법적 구속력 없음 (소송 전 예고 성격)

### 경찰 출석 요구 받았을 때
1. 출석일시 확인 (통상 7-14일 여유)
2. **자진 출석 전 변호사 상담 필수**
3. 진술서·각서 절대 혼자 작성 금지
4. 피의자 신분이면 묵비권 고지 받을 권리 있음

### 고소 협박 받았을 때
1. 협박 자체는 범죄가 될 수 있음 (협박죄, 공갈죄 검토)
2. 모든 대화 캡처·보존
3. 절대 감정적 반응 금지
4. 상대방의 주장 내용을 정확히 파악

### 저작권 침해 통보 받았을 때
1. 플랫폼별 이의신청 기간 확인 (유튜브: 영업일 기준)
2. 사용한 음악·영상의 출처 확인
3. 라이선스 문서 수집
