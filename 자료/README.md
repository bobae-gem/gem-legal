# 자료 폴더

계약서 템플릿, 법령 자료, 판례 정리, 내용증명 양식 등을 저장한다.

## 폴더 구조 (예정)

```
자료/
├── 계약서템플릿/
│   ├── 협찬계약서_기본형.md
│   ├── 협찬계약서_단기형.md
│   └── 촬영동의서.md
├── 내용증명/
│   ├── 내용증명_발송용.md
│   └── 내용증명_대응용.md
├── 법령/
│   ├── 저작권법_핵심조문.md
│   ├── 개인정보보호법_핵심.md
│   └── 표시광고법_협찬공시.md
└── 판례/
    ├── 댄스커버_저작권판례.md
    └── SNS_명예훼손판례.md
```

## 자료 추가 방법

사건을 처리하면서 재사용 가능한 자료는 이 폴더에 저장한다.
