# 참고자료: 사건분석·증거정리 양식
> 구 2단계 사건분석팀 자료. 증거 분류표, 타임라인, 쟁점 정리 양식 참고용.

## 사실관계 타임라인 양식

```
[날짜] [사건] [증거 有/無] [비고]
YYYY-MM-DD | 사건 내용 | ✅/❌ | 중요도 (상/중/하)
```

- 날짜 불분명한 사실 → "추정: " 표시
- 양측 주장이 다른 부분 → 🔴 표시
- 증거 없는 부분 → ❌ 표시 후 수집 방법 제안

## 디지털 증거 보전 방법

1. **스크린샷**: 날짜·시간 포함, URL 함께 캡처
2. **카카오톡**: 내보내기 기능 → txt 파일 저장
3. **이메일**: 헤더 포함 전체 저장 (eml 형식)
4. **유튜브·SNS**: 공증 또는 화면녹화 + 타임스탬프
5. **공정한 제3자 캡처**: 화면공증 서비스 활용 가능

### 증거 삭제 방지 긴급 조치
- 상대방이 증거를 삭제할 우려 → **증거보전신청** (법원)
- 플랫폼 데이터 보존 요청 가능 (일부 플랫폼)
- 공증사무소 화면공증 (비용 발생)

## 보배 채널 특수 증거

| 상황 | 필요 증거 |
|------|-----------|
| 저작권 분쟁 | 원본 파일 생성일, 업로드 기록, 편집 히스토리 |
| 협찬 계약 분쟁 | 계약서, 카톡 협의내용, 정산 내역, 영상 공개일 |
| 명예훼손 | 해당 게시물, 조회수·조회기간, 제3자 반응 |
| 초상권 침해 | 촬영 동의 여부, 동의서, 관계 증명 |
| 촬영 허가 분쟁 | 허가 신청서, 장소 사용 계약, 현장 안내 |

## 사건 유형별 핵심 쟁점

### 저작권 침해
- 원저작물과 침해 저작물의 실질적 유사성
- 의거성 (상대방이 원저작물에 접근했는가)
- 공정이용 해당 여부
- 손해액 산정 방법 (실제 손해 vs 법정 손해배상)

### 명예훼손
- 적시된 사실의 구체성
- 공연성 (불특정 다수에게 알려질 가능성)
- 진실성 및 공공의 이익 (형법 310조 위법성 조각)
- 허위사실 여부

### 계약 분쟁
- 계약 성립 여부 (청약·승낙)
- 채무불이행의 귀책사유
- 손해 발생 및 인과관계
- 위약금 조항의 유효성
