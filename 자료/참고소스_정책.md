## 소스 1: 국가법령정보센터 (필수)

**URL**: https://www.law.go.kr

**역할**: 현재 실제 시행 중인 법령 확인. 조문 번호까지 명시.

**사용 법령 목록**:

| 법령 | 주요 적용 상황 |
|------|--------------|
| 민법 | 불법행위(750조), 손해배상(390조), 위자료 |
| 형법 | 명예훼손(307조), 모욕(311조), 협박(283조) |
| 정보통신망법 | 온라인 명예훼손(70조), 사이버 모욕 |
| 개인정보보호법 | 초상권, 개인정보 무단 공개 |
| 민사소송법 | 소송 절차, 증거 제출, 기일 |
| 형사소송법 | 고소, 수사, 공판 절차 |
| 저작권법 | 저작물 침해(136조), 손해배상(125조), 공정이용 |
| 표시광고법 | 협찬 공시 의무 |
| 아동복지법 | 아동 출연 콘텐츠 |

**사용 규칙**:
- 조문 인용 시 반드시 법령명 + 조항 번호 명시
- "~라고 들었다" 수준의 법 설명 금지
- 개정 여부 확인 후 최신 조문 기준 적용

## 소스 2: 대법원 종합법률정보

**URL**: https://glaw.scourt.go.kr

**역할**: 실제 판례 검색 및 인용. 법리 근거 확보.

**판례 계층**:
```
대법원 판결 > 고등법원 판결 > 지방법원 판결
(구속력 순서 — 대법원이 가장 강함)
```

**보배 법률전략실 핵심 판례 영역**:

| 쟁점 | 검색 키워드 |
|------|------------|
| 유튜브 명예훼손 | "유튜브" "명예훼손" "공연성" |
| 해명 방송 법적 성격 | "해명" "방어 목적" "위법성 조각" |
| SNS 모욕죄 | "SNS" "모욕" "특정성" "공연성" |
| 초상권 침해 | "초상권" "동의 없이" "위자료" |
| 스트리머 손해배상 | "크리에이터" "유튜버" "손해배상" |
| 사생활 공개 | "사생활" "사적 영역" "공개" "위자료" |
| 위자료 산정 기준 | "위자료" "산정 기준" "온라인" |

**판례 인용 형식**:
```
대법원 YYYY. M. D. 선고 YYYYXX다XXXXXX 판결
```

**사용 규칙**:
- 판례 인용 시 사건번호 반드시 포함
- "~한다는 판례가 있다" 수준 금지 → 구체적 사건번호 제시
- 하급심 판례는 "(참고)" 표시, 대법원 판례와 구분

## 소스 3: 판결문 인터넷 열람

**URL**: https://www.scourt.go.kr (대법원 전자소송)
**서비스**: 법원 판결문 공개 열람 서비스

**역할**: 실제 판결문 원문 확인. 법원이 인정한 사실과 이유를 직접 확인.

**특히 유용한 경우**:
- 항소이유서 작성 전 1심 판결문 분석
- 유사 사건 판결 이유 검토
- 법원이 어떤 증거를 채택/배척했는지 확인

**주의**:
- 개인정보 비실명 처리된 판결문이 대부분
- 최근 판결일수록 공개 제한 있을 수 있음

## 소스 4: 플랫폼 정책

**역할**: 법적으로 문제없어도 플랫폼 신고·제재가 가능하므로 병행 확인 필수.

### 유튜브
- **커뮤니티 가이드**: https://www.youtube.com/howyoutubeworks/policies/community-guidelines/
- **저작권 정책**: Content ID, DMCA 이의신청 절차
- **광고 정책**: 협찬 공시 의무 (유료 프로모션 체크박스)
- **핵심 위험**: 저작권 경고 3회 → 채널 삭제

### 틱톡
- **커뮤니티 가이드**: https://www.tiktok.com/community-guidelines/
- **라이브 정책**: 라이브 중 음악 재생 → 즉시 경고
- **핵심 위험**: 실시간 신고 누적 → 라이브 차단

### 인스타그램
- **커뮤니티 가이드**: https://help.instagram.com/477434105621119
- **광고 공시**: #광고 #협찬 #AD 표시 위치
- **핵심 위험**: 반복 신고 → 계정 일시 비활성화

### 네이버·카카오
- 플랫폼 자체 신고 시스템 → 명예훼손성 콘텐츠 블라인드

**플랫폼 vs 법 비교 원칙**:
```
법적으로 허용 + 플랫폼 위반 → 채널 제재 가능 (법과 무관)
법적으로 위반 + 플랫폼 허용 → 법적 고소 가능 (플랫폼과 무관)
두 기준 모두 확인 필수
```

## 실패 사례 DB (반드시 참고)

법적 실수를 반복하지 않기 위해 아래 실패 패턴을 항상 확인한다.

### 스트리머·크리에이터 주요 실패 사례

| 실패 유형 | 내용 | 법적 결과 |
|-----------|------|-----------|
| 감정 라이브 후 기사화 | 분노 상태 라이브 → 단정 발언 → 언론 보도 | 명예훼손 고소 + 2차 피해 |
| 댓글 때문에 역고소 | 상대방 댓글에 흥분한 답글 → 협박·모욕 역고소 | 형사 역고소 자료 제공 |
| 저격 썸네일 때문에 악의 인정 | 특정인 지목 썸네일 → 고의성 증거로 사용 | 위자료 증액 (악의적 행위 인정) |
| 사생활 공개로 위자료 증가 | 합의 불가 내용을 방송에서 공개 | 위자료 2~3배 증가 |
| 팬 공격 교사로 공범 인정 | "여러분이 판단하세요" 발언 → 팬 공격 → 교사죄 | 공범 책임 |
| 해명 영상에서 자백 | 해명하다가 불리한 사실 인정 발언 | 법정 증거로 제출됨 |
| 증거 삭제 후 인멸죄 | 불리한 채팅 삭제 → 증거인멸 혐의 | 형사 가중 처벌 |
| SNS 스토리 역이용 | 분노 암시 스토리 → 상대방이 캡처 → 협박 증거 | 역고소 자료 |
