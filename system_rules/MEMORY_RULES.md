# 메모리 관리 규칙

> GLOBAL_RULES.md 우선 적용.

---

## 저장공간 3구역 엄격 분리

| 공간 | 위치 | 저장 내용 | 핵심 규칙 |
|------|------|-----------|-----------|
| 시스템 | `저장공간/시스템/` | 구조·역할·원칙만 | 사건 정보 오염 절대 금지 |
| 사건 | `저장공간/사건/사건명/` | 판결문·증거·타임라인 | 사건별 완전 분리, 승인 후 저장 |
| 임시 | `저장공간/임시/` | 감정·추측·미확정 | 자동 장기저장 금지 |

---

## 저장 금지 항목

1. **시스템 공간에 사건 정보 저장 금지**
2. **사건 A 정보를 사건 B 공간에 저장 금지**
3. **감정 표현을 사실처럼 저장 금지**
4. **임시 내용을 승인 없이 사건 공간으로 이동 금지**
5. **추측성 분석을 확정 사실로 저장 금지**

---

## 임시 저장공간 규칙

- 감정 메모 / 아이디어 / 추측 / 미확인 정보
- **3일 이상 경과 시 자동 보류 검토**
- 사용자 승인 없이 장기 저장 금지
- 임시 → 사건 공간 이동은 반드시 사용자 승인 필요

---

## 사건별 메모리 혼합 금지

- 사건 A에서 수집된 상대방 정보를 사건 B에 적용 금지
- 서로 다른 사건의 증거·주장 혼합 금지
- 각 사건은 독립된 메모리 공간에서만 처리

## dashboard 저장 구조

| 파일 | 용도 | 규칙 |
|------|------|------|
| `dashboard_data.json` | 확정 정보 | 승인된 것만 |
| `dashboard_pending.json` | 검토 대기 | 분류실 거쳐야 |
| `dashboard_log.json` | 변경 기록 | 모든 변경 기록 |
| `raw_monitoring.json` | 수집 원본 | 분류 전 임시 |
