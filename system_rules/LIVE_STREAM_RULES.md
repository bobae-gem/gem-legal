# 라이브 방송 규칙

> GLOBAL_RULES.md 우선 적용.
> 실시간 방송은 고위험 영역으로 분류된다.

---

## 핵심 원칙

라이브 방송 중 발언은:
- **취소 불가** (실시간 증거 생성)
- **확산 속도 매우 빠름**
- **클립 캡처 즉시 가능**
- **공연성 자동 충족** (시청자 수)

→ 분쟁 중 라이브는 최고 위험 행동으로 분류

---

## 라이브 전 필수 체크

| 항목 | 확인 |
|------|------|
| 감정 상태 안정 여부 | □ |
| 사건 관련 언급 계획 없음 | □ |
| 상대방 언급 계획 없음 | □ |
| 허위사실 언급 가능성 없음 | □ |
| 팬 선동 가능성 없음 | □ |
| 사생활 공개 가능성 없음 | □ |
| 콘텐츠 검수실 검토 완료 | □ |
| 행동위험차단팀 검토 완료 | □ |

**하나라도 체크 안 되면 라이브 연기 강력 권고**

---

## 라이브 중 절대 금지

1. 분쟁 상대 직접 언급
2. 사건 경위 설명 (해명 포함)
3. 감정적 발언 ("억울하다", "화난다" 이상의 표현)
4. 상대방 행동 단정 ("저 사람이 ~했다")
5. 팬에게 행동 암시 ("여러분이 판단하세요")
6. 사생활 정보 공개
7. 법원 사건 관련 발언

---

## 라이브 위험 패턴

### 감정 라이브 패턴
분노 상태 → 단정 발언 → 클립 캡처 → 기사화 → 고소

### 해명 과잉 패턴
해명 의도 → 추가 사실 공개 → 새 명예훼손 성립

### 팬덤 선동 패턴
암시적 발언 → 팬 공격 → 교사죄 공범

---

## 라이브 중 문제 발생 시

1. 즉시 라이브 종료
2. 행동위험차단팀에 즉시 보고
3. 추가 대응 전 법률전략실 검토
4. 삭제·편집 전 증거보존팀 캡처 확인

---

## 라이브 종료 후 체크

- 문제 발언 여부 확인
- 클립 확산 여부 모니터링
- 댓글 반응 분석
- 필요시 증거보존 즉시 실행
